# usbgadget

USB gadget stack for Raspberry Pi: HID keyboard/mouse + CDC ACM + UVC.

## Repo layout

- scripts: gadget setup and streaming logic
- configs: default config files
- services: systemd service unit files

See DEVLOG.md for development steps and current progress.
